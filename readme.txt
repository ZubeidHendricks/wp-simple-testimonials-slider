Show rotating customer testimonials with a simple shortcode — names, roles and star ratings, no jQuery.

== Description ==

Add social proof anywhere with an auto-rotating testimonials slider. Wrap your
quotes in `[testimonials]` and add a `[testimonial]` for each, with a name, role
and optional star rating. Auto-rotates, pauses on hover, and has clickable dots.

    [testimonials]
      [testimonial name="Jane Doe" role="CEO, Acme" stars="5"]Best decision we made.[/testimonial]
      [testimonial name="Sam Lee" role="Founder" stars="5"]Support is incredible.[/testimonial]
    [testimonials]

**Free features**

* Auto-rotating slider with dots and pause-on-hover.
* Names, roles and 0–5 star ratings.
* Custom accent colour and rotation speed.
* No jQuery, accessible markup.

**Pro features**

* Avatars/photos and company logos.
* Grid layout and import from reviews.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Add the [testimonials] / [testimonial] shortcodes to any page.
