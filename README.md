# Testimonials Slider

Show rotating customer testimonials with a simple shortcode — names, roles and
star ratings, no jQuery. Part of the
[Zub Plugin Factory](https://github.com/ZubeidHendricks/wp-plugin-factory).

## What it does

```
[testimonials]
  [testimonial name="Jane Doe" role="CEO, Acme" stars="5"]Best decision we made.[/testimonial]
  [testimonial name="Sam Lee"  role="Founder"   stars="5"]Support is incredible.[/testimonial]
[testimonials]
```

Auto-rotating slider with clickable dots, pause-on-hover, and 0–5 star ratings.
Configurable accent colour and rotation speed.

## Settings

**Settings → Testimonials Slider** — accent colour, auto-rotate interval.
